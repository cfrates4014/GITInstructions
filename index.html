GitHub Set-up Instructions

1) First if you don’t have a GitHub page yet you will need to sign up. It’s easy and free. So please navigate to https://github.com .

2) Once you have signed and signed in click “New Repository” look for the + sign top right hand corner of page..

3) On the repository page it will now ask you to name your repository. You will want to name it your “username”.github.io Example: username.github.io

4) To begin setting up your repository open up your “settings” tab within your repository. <> <
